# image-classifier

This project implements a binary image classification model that recognizes whether an image contains a dog or a cat using transfer learning with a pretrained convolutional neural network.

The model is built with TensorFlow/Keras and evaluated using multiple classification metrics across different train–test splits.

# Project Overview

* The main goals of this project are:

* Build a binary image classifier (Dog vs Cat)

* Apply transfer learning

* Analyze the impact of different test set sizes

* Evaluate the model using multiple performance metrics

* The model is based on MobileNetV2, pretrained on ImageNet.
